Sudoku Solver is an Android Studio project written in Java and XML which can solve any Sudoku game with the help of "Recursive Backtracking" algorithm.
Capture files displays how the app looks on Google Pixel 2.

Following are the most important files of the project:

1. MainActivity.java (/app/src/main/java/com/example/sudokusolver/MainActivity.java): This is the Java file which acts as the Backend for the application
   and includes implementation of Recrsive Backtracking algorithm to find the possible solution of the game.

2. activity_main.xml (/app/src/main/res/layout/activity_main.xml): This is the layout file written in XML which acts as the Frontend of the application.
